Asking Simple Questions En Francais

Last updated Unknown

Table of Contents

  • Asking questions in French
    • Quand tu veux poser un question, tu ne dois que utiliser les trois formes dessous
      • Tu travailles aujourd’hui? ( Just using a question mark )
      • Est-ce que tu travailles aujourd’hui? ( Using Est-ce que )
      • Travailles-tu aujourd’hui? ( Verb inversion )
  • We can create more open-ended questions by adding in more question words
    • Qui ( Who )
      • Qui est-ce que tu aimes?
      • Qui est-ce que tu baises, epouses et tues
    • Avec Qui ( With whom ? )
      • Avec qui est-ce que tu rigoles le plus?
    • Quand
      • Quand est-ce que tu rigoles le plus?
    • A quelle heure
      • A quelle heure es-ce que vous pensez arriver?
    • Qu’est-ce que ( what are you … )
      • Qu’est-ce que tu utilises comme site pour apprendre le francais
      • Qu’est que tu manges? ( What are u eating )
    • Comment ( How )
      • Comment est-ce qu’on utilise ce truc? ( How do you use this thing )
    • Combien ( How many )
      • Combien de fois par mois est-ce qu’ils gagnent au loto ?
    • Where
      • Où est-ce que tu achètes tes légumes bios
    • Pourquoi
      • Pourquoi est-ce que tu fumes? (Why do you smoke? )
  • Using Ca, Si and Non for emphasis
    • Ca : Using this just at the end
      • Où est-ce que tu habites ? => Oui, mais où ça ? ( Where is this? )
      • Le voyage ne dure que deux heures! => Comment ça ?
    • Non ( For positive question )
      • Vous êtes américain, non/n’est-ce-pas ( You’re american , right? )
    • Si ( For negative questions )
      • Vous n’êtes pas américain, si
  • T-il ( When we use an inverted question, the il/elle/on requires an extra t for the sound to carry over )
    • Mange-t-il de la viande? instead of Mange-il de la viande ?

Words

  • Est-ce qu’il faut repeter a chaque fois?
  • Est-ce qu’on voyage cet ete?
  • J’ai mal a la gorge
  • Ca c’est suffisant pour la maintenant ( ok, this is enoguh for the moment )
  • On va continuer dans un instant ( We can continue in an instant )
  • Comment ? (what? Don’t use Quoi, which is more aggressive )
  • Où est-ce que tu comptes voyager cette année ? ( Where do you plan to travel this year? )
  • Combien de fois par an est-ce que vous voyagez ensemble ? ( How many times a year do you travel together)
  • Avec qui est-ce que tu voyages demain ( With who are you travelling with? )
  • Bon courage ( Good Luck )
  • au cas (In case, eg. Prepare cinq questions en avance pour éviter des moments de silence au cas où ils arrivent )
  • Je me leve a 8 heures ( I wake up at 8 o clock )
  • Tu passes combien de temps avec ta famille? ( How much time do you spend with your family )
  • Qu’est-ce que tu regardes a la tele quand il n’y a personne autour ( What do you watch when nobody is around? )
  • Pourquoi est-ce que tu traites ta maman comme ca
  • Depuis quand est-ce que vous habitez ensemble?
  • Penses-tu changer ton comportement
  • Comment ca marche ce truc ( How does this thing work )
  • Quand es-ce que vous avez demenage a Paris ( When did you move to Paris? )
  • C’est combien le loyer? ( How much is the rent )
  • Comment est-ce que vous vous etes recontres? ( How did you guys meet )
  • Bah pourquoi pas? ( Why not )
  • Est-ce que tu t’amuses ( Are you having fun? )
